API ini digunakan untuk mendapatkan waktu sholat berdasarkan lokasi, latitude, longitude, dan tanggal.

ðŸ”¹ Endpoint

GET http://localhost/api-waktu-sholat/public/index.php

ðŸ”¹ Parameter

  • Anda bisa memilih menggunakan lokasi atau latitude dan longitude:
  • Jika menggunakan lokasi:
    • lokasi (string) - Nama lokasi (opsional, jika tidak menggunakan latitude & longitude)
  • Jika menggunakan latitude dan longitude:
    • latitude (float) - Koordinat lintang (opsional)
    • longitude (float) - Koordinat bujur (opsional)
  • tanggal (YYYY-MM-DD) - Format: tahun-bulan-hari (wajib diisi)

GET http://localhost/api-waktu-sholat/public/index.php?lokasi=Jakarta&tanggal=2025-02-10
